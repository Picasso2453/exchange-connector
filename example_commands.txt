L2 Orderbook (SOL):
dotnet run --project src/Connector.Cli -- --exchange hl --symbols SOL --channels l2 --no-auth

Candles (BTC, 1m interval):
dotnet run --project src/Connector.Cli -- --exchange hl --symbols BTC --channels candles --no-auth

Both together (BTC + ETH, trades + L2 + candles):
dotnet run --project src/Connector.Cli -- --exchange hl --symbols BTC,ETH --channels trades,l2,candles --no-auth

With raw exchange payloads included:
dotnet run --project src/Connector.Cli -- --exchange hl --symbols SOL --channels l2,candles --no-auth --raw
