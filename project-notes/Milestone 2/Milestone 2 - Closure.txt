Shipped scope
  - MEXC spot trades adapter (protobuf decode) with futures endpoint scaffold.
  - Mux command: subscribe trades --sub ... with envelope JSONL output.
  - Legacy HL raw output preserved via --format raw.
  - Best-effort mux policy for partial connectivity.
  - Offline protobuf decode test.

Best-effort mux policy (M2)
  - If a source fails to connect, log to stderr and continue with others.
  - Exit 0 if stop condition met and at least one envelope line emitted.
  - Exit 1 if zero envelopes emitted.

Key commands
  - HL legacy raw:
    dotnet run --project src/xws -- hl subscribe trades --symbol SOL --format raw --max-messages 10 --timeout-seconds 30

  - MEXC spot trades:
    dotnet run --project src/xws -- mexc spot subscribe trades --symbol BTCUSDT --max-messages 10 --timeout-seconds 30

  - Mux HL + MEXC:
    dotnet run --project src/xws -- subscribe trades --sub hl=SOL --sub mexc.spot=BTCUSDT --max-messages 50 --timeout-seconds 30

Notes
  - MEXC WS may be blocked regionally; code handles this as best-effort.
  - Smoke scripts accept XWS_SMOKE_ALLOW_NETFAIL=1 or XWS_MEXC_ALLOW_FAIL=1.

Validation
  - dotnet build -c Release
  - dotnet test -c Release
  - scripts/smoke-m2.ps1 (with XWS_SMOKE_ALLOW_NETFAIL=1 on restricted networks)
